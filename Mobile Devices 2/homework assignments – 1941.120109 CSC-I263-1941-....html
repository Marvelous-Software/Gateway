        • assignment 1

          Write a program in Kotlin that

          1. declares 2 variables
            • var a = 3
            • var b = 5
          2. prints the values of the variables
          3. swap the values in the variables
          4. print the values of the variables again

          What to submit

          • source code in Kotlin
          • a sample of the output (e.g. screenshot)


          IF in class you used solutions such as 

          a = b.also { b = a }

          THEN you also need to submit a solution that uses that approach along with an explanation of how the function also works.

          • Kotlin code (i.e ..kt file)
          • examples of output
