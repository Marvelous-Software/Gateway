          Option 2 - project based on MySQL database:

          Using the attached database

          , create a Web interface based on PHP for the following 2 users:

          1. administrator. They should be able to insert/add/delete records for tables song, artist
          2. user. They should be able to search and and add/delete records for playlists

          Access to the Web interface should be managed by a login

          This project is broken down into 3 milsteones (see below for detailes along with deadlines). Submit all milestones into this assignment (i.e. as submissions). The grade will be an average of all the submissions.

          Milestone 1 (DUE: 11-22-2020)


          Option 1

          Implement a class "Asteroid" and create objects to store all the neo (e.g. from https://api.nasa.gov/neo/rest/v1/feed?start_date=2018-08-01&end_date=2018-08-04&api_key=gx1daXuj2m8vbtrVKtHALsIVu8U2yvBOWcbyjch8)

          The class should store only the information that you are going to display.

          Implement the "summary" Web page where you are going to show all the neos


          Option 2

          Implent the classes "Song" and "Artist" and create the "administrator" Web pages of the application.



          Milestone 2 (DUE: 12-02-2020)


          Option 1

          Create the "details" Web pagwhere you are going to show all the details per each object


          Option 2

          Implent the "user" Web pages of the application.



          Milestone 3 (DUE: 12-06-2020)


          Both option 1 and 2: password protect all the pages you created by implementing an authenticating mechanism
